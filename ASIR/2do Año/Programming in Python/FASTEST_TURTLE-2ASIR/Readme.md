# Algoritmia Gráfica y Vectorial con Python Turtle

- **Descripción:** Proyecto de programación creativa enfocado en el uso de la librería gráfica turtle para la generación de gráficos vectoriales mediante código. El objetivo es dominar el sistema de coordenadas cartesianas y el control de flujo para dibujar formas geométricas complejas y tipografía manual sin depender de fuentes preinstaladas.

- **Tecnologías:** Python 3, Turtle Graphics, Tkinter.

- **Funcionalidades Clave:**
    - **Renderizado Geométrico (Logotipo):** Implementación algorítmica para dibujar la estrella institucional de La Salle, calculando ángulos y longitudes de trazo precisas.
    - **Tipografía Vectorial Manual:** Script para "escribir" el nombre del usuario trazo a trazo (simulando un plotter). Se ha desarrollado la lógica de cada letra manualmente, ya que el uso de la función nativa write() estaba restringido por requerimientos técnicos.
    - **Arte Procedural (Generativo)** Creación de un patrón gráfico complejo mediante bucles (for/while) que ejecutan más de 30 operaciones de dibujo consecutivas, demostrando el poder de la automatización en el diseño visual.

- **Aprendizaje:** Manejo del plano cartesiano (X, Y), trigonometría básica para rotaciones (left/right), y uso de bucles para la generación de patrones repetitivos y fractales.
